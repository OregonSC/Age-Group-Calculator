# Soccer Age Group Calculator

Simple web-based calculator for determining soccer age group based on:

- Current school grade
- Birthday

Includes:
- August flexibility rule
- Grade-to-U mapping
- Age mismatch helper warning

## Hosting

This project is designed to be hosted on GitHub Pages.

### To deploy:

1. Push this repo to GitHub
2. Go to Settings
3. Click Pages
4. Select branch `main`
5. Save

Your app will be live at:

https://oregonsc.github.io/Age-Group-Calculator/
